Welcome to the Getting Started guide! This section will help you understand the fundamentals of the Better Notion SDK.

## What You'll Learn

- **Installation**: Set up the SDK in your project
- **Authentication**: Configure access to Notion
- **Basic Concepts**: Understand the entity model
- **Project Setup**: Best practices for structuring your code

## Prerequisites

Before you begin, ensure you have:

- Python 3.10 or higher installed
- pip or uv for package management
- A Notion account with integration access
- Basic knowledge of async/await in Python

<Callout type="info">
  **New to async/await?** Check out the [Advanced - Async Patterns](../advanced/async-patterns) guide for a refresher.
</Callout>

## Learning Path

We recommend following these guides in order:

1. **Installation** - Get the SDK installed
2. **Authentication** - Set up your Notion integration
3. **Basic Concepts** - Learn how the SDK is structured
4. **Project Setup** - Configure your project properly

After completing these guides, you can explore:

- **[Guides](../guides)** - Task-based tutorials for specific operations
- **[API Reference](../api)** - Complete technical documentation
- **[Examples](../examples)** - Real-world projects
