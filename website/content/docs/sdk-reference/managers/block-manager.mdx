# BlockManager

Manages **Block** entities with create, update, and delete operations.

## Generic Operations

<Blocks /> ### get()

Retrieve a block by ID:

```python
block = await client.blocks.get("block_id")
```

**Returns:** Specialized `Block` object (Code, Todo, Paragraph, etc.)

**Example:**

```python
block = await client.blocks.get("block-123")

if block.is_paragraph:
    print(f"Text: {block.text}")
elif block.is_todo:
    print(f"Todo: {block.checked}")
```

<Edit /> ### update()

Update a block:

```python
updated = await client.blocks.update(
    block,
    paragraph={"rich_text": [{"type": "text", "text": {"content": "New text"}}]}
)
```

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `block` | `Block` | Yes | Block to update |
| `**kwargs` | `dict` | Yes | Block properties to update |

**Returns:** Updated `Block` object

**Example:**

```python
# Update paragraph
block = await client.blocks.get("block_id")
updated = await client.blocks.update(
    block,
    paragraph={"rich_text": [...]}
)

# Update todo
todo = await client.blocks.get("todo_id")
updated = await client.blocks.update(
    todo,
    to_do={
        "checked": True,
        "rich_text": todo._data.get("to_do", {}).get("rich_text", [])
    }
)
```

<Trash2 /> ### delete()

Delete a block:

```python
await client.blocks.delete(block)
```

**Example:**

```python
block = await client.blocks.get("block_id")
await client.blocks.delete(block)
```

## Complete Example

```python
import asyncio
from better_notion import NotionClient

async def create_document():
    """Create a document with various blocks."""
    client = NotionClient(auth="secret_...")

    # Get parent page
    page = await client.pages.get("page_id")

    # Create heading
    h1 = await client.blocks.create_heading(
        parent=page,
        text="Introduction",
        level=1
    )

    # Create paragraph
    para = await client.blocks.create_paragraph(
        parent=page,
        text="This is the introduction."
    )

    # Create todo
    todo = await client.blocks.create_todo(
        parent=page,
        text="Review document",
        checked=False
    )

    # Update todo
    await todo.set_checked(True)

    # Create code block
    code = await client.blocks.create_code(
        parent=page,
        code="print('Hello, world!')",
        language="python"
    )

    # Create list
    for i in range(3):
        await client.blocks.create_bullet(
            parent=page,
            text=f"Point {i+1}"
        )

    # Create quote
    await client.blocks.create_quote(
        parent=page,
        text="Simplicity is the ultimate sophistication"
    )

    # Create divider
    await client.blocks.create_divider(parent=page)

    print(f"Created document with blocks")

asyncio.run(create_document())
```
