# Page Model

The `Page` model represents Notion pages with rich properties and autonomous CRUD operations.

## Constructor

```python
from better_notion._sdk.models import Page

page = Page(client, data)
```

Typically created via:
- `Page.get()` - Retrieve existing page
- `Page.create()` - Create new page
- `Page.from_data()` - Create from API data

## Properties

<FileText /> ### title

Page title (derived from title property).

```python
page = await Page.get("page_id", client=client)
print(page.title)
```

**Type:** `str | None`

<Link /> ### url

Notion page URL.

```python
print(page.url)
# https://notion.so/page-id-encoded
```

**Type:** `str`

### icon

Page icon (emoji or image URL).

```python
icon = page.icon
print(icon)  # "ðŸš€" or "https://..."
```

**Type:** `str | None`

### cover

Page cover image URL.

```python
cover = page.cover
print(cover)  # "https://..."
```

**Type:** `str | None`

<EyeOff /> ### archived

Archive status.

```python
if page.archived:
    print("This page is archived")
```

**Type:** `bool`

### parent

Parent object data.

```python
parent_type = page.parent["type"]  # "database_id", "page_id", etc.
parent_id = page.parent.get("database_id") or page.parent.get("page_id")
```

**Type:** `dict`

### created_time, last_edited_time

ISO 8601 timestamps.

```python
from datetime import datetime

created = datetime.fromisoformat(page.created_time)
edited = datetime.fromisoformat(page.last_edited_time)
```

**Type:** `str`

## Class Methods

### get()

Retrieve a page by ID (uses cache).

```python
page = await Page.get("page_id", client=client)
```

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page_id` | `str` | Yes | Page UUID |
| `client` | `NotionClient` | Yes | Client instance |

**Returns:** `Page` object

### create()

Create a new page.

```python
page = await Page.create(
    parent=database,
    client=client,
    title="My Page",
    **properties
)
```

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `parent` | `Database` or `Page` | Yes | Parent object |
| `client` | `NotionClient` | Yes | Client instance |
| `title` | `str` | No | Page title |
| `**kwargs` | `any` | No | Additional properties (icon, cover, etc.) |

**Returns:** Newly created `Page` object

**Example:**

```python
# Simple page
page = await Page.create(
    parent=database,
    client=client,
    title="New Task"
)

# With properties
page = await Page.create(
    parent=database,
    client=client,
    title="Project X",
    status="In Progress",
    priority="High"
)

# With icon and cover
page = await Page.create(
    parent=database,
    client=client,
    title="Important Page",
    icon=Page.format_icon(emoji="ðŸš€"),
    cover=Page.format_cover(external_url="https://example.com/img.png")
)
```

## Instance Methods

### get_property()

Get a property value by name.

```python
status = page.get_property("Status")
tags = page.get_property("Tags")
```

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | `str` | Yes | Property name |

**Returns:** Property value (type varies)

### has_property()

Check if property exists.

```python
if page.has_property("Priority"):
    priority = page.get_property("Priority")
```

### parent()

Get parent object.

```python
parent = await page.parent()

if isinstance(parent, Database):
    print(f"In database: {parent.title}")
```

**Returns:** `Database`, `Page`, or `Workspace` object

### children()

Iterate over child blocks.

```python
async for block in page.children():
    print(f"{block.type}: {block}")
```

**Returns:** Async iterator of `Block` objects

## Use Cases

### Create Page with Icon/Cover

```python
page = await Page.create(
    parent=database,
    client=client,
    title="Launch Page",
    icon=Page.format_icon(emoji="ðŸš€"),
    cover=Page.format_cover(external_url="https://example.com/banner.png")
)
```

### Access Custom Properties

```python
# Safe property access
status = page.get_property("Status") if page.has_property("Status") else None

# Get multiple properties
properties = {
    "Status": page.get_property("Status"),
    "Priority": page.get_property("Priority"),
    "Assignee": page.get_property("Assignee")
}
```

### Navigate Hierarchy

```python
# Get parent database
parent = await page.parent()
if isinstance(parent, Database):
    print(f"Database: {parent.title}")
    print(f"Properties: {list(parent.schema.keys())}")

# Iterate children
async for block in page.children():
    if block.is_paragraph:
        print(block.text)
```
